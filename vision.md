# Infrastructure

## Platform

- MultiPlatform - mainly Mobile App
- mono-repo codebase
    - one build for each platform
    - one landing page - github pages first then can also be hosted in a separate domain

## Data

### Persistence

- Local Storage

- Networking is hard and slow

### Format

- simple authoring, rich visual rendering (markdown-like simplicity, not markdown literally)
- memes and gifs are data?

### Sharing

- Export as .pensine file and share through any channel (messaging apps, email, AirDrop)
- No backend needed, doubles as backup/export format
